### 홈 화면과 레이아웃 설정

- **홈 컨트롤러 등록**: `HomeController`는 루트 URL("/")에 매핑되어 있으며, "home" 문자열을 반환하여 홈 화면을 표시함
- **타임리프 설정**: 스프링 부트에서 타임리프를 사용하여 뷰를 렌더링하며, 템플릿 파일은 `classpath:/templates/` 경로에 위치함
- **HTML 구조**: `templates/home.html` 파일은 타임리프를 사용하여 헤더, 바디, 푸터를 포함한 기본 레이아웃을 정의함

### 회원 등록

- **회원 등록 폼 객체**: `MemberForm` 클래스는 회원의 이름, 도시, 거리, 우편번호를 포함한 정보를 저장함
- **회원 등록 컨트롤러**: `MemberController`는 회원 등록 폼을 표시하고, 폼 제출 시 회원 정보를 저장함
    - `createForm` 메서드는 새로운 `MemberForm` 객체를 모델에 추가하여 회원 등록 폼을 표시함
    - `create` 메서드는 폼 데이터를 검증하고, 유효한 경우 회원 정보를 저장한 후 홈 화면으로 리다이렉트함
    

### 회원 목록 조회

- **회원 목록 컨트롤러**: `MemberController`는 회원 목록을 조회하여 뷰에 전달함
    - `list` 메서드는 `memberService`를 통해 회원 목록을 조회하고, 이를 모델에 추가하여 `memberList` 뷰를 반환함
    

### 상품 등록

- **상품 등록 폼 객체**: `BookForm` 클래스는 상품의 이름, 가격, 수량, 저자, ISBN 정보를 저장함
- **상품 등록 컨트롤러**: `ItemController`는 상품 등록 폼을 표시하고, 폼 제출 시 상품 정보를 저장함
    - `createForm` 메서드는 새로운 `BookForm` 객체를 모델에 추가하여 상품 등록 폼을 표시함
    - `create` 메서드는 폼 데이터를 사용하여 새로운 `Book` 객체를 생성하고 저장한 후 상품 목록으로 리다이렉트함
    

### 상품 목록 조회

- **상품 목록 컨트롤러**: `ItemController`는 상품 목록을 조회하여 뷰에 전달함
    - `list` 메서드는 `itemService`를 통해 상품 목록을 조회하고, 이를 모델에 추가하여 `itemList` 뷰를 반환함
    

### 상품 수정

- **상품 수정 컨트롤러**: `ItemController`는 상품 수정 폼을 표시하고, 폼 제출 시 상품 정보를 수정함
    - `updateItemForm` 메서드는 수정할 상품 정보를 조회하여 `BookForm` 객체에 설정하고, 이를 모델에 추가하여 수정 폼을 표시함
    - `updateItem` 메서드는 폼 데이터를 사용하여 상품 정보를 수정하고 저장한 후 상품 목록으로 리다이렉트함
    

### 주문 기능

- **주문 컨트롤러**: `OrderController`는 주문 폼을 표시하고, 주문을 처리하며, 주문 목록을 조회하고 취소함
    - `createForm` 메서드는 회원과 상품 목록을 조회하여 모델에 추가하고, 주문 폼을 표시함
    - `order` 메서드는 주문 정보를 받아 주문을 처리하고, 주문 목록으로 리다이렉트함
    - `orderList` 메서드는 주문 목록을 조회하여 모델에 추가하고, 주문 목록 뷰를 반환함
    - `cancelOrder` 메서드는 주문을 취소하고, 주문 목록으로 리다이렉트함
